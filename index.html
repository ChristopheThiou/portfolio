Projet Immobillier

Ce projet immobilier a été développé dans le cadre de ma formation en développement web. Il s'agit d'une plateforme en ligne qui permet aux agences immobilières de gérer leurs biens. Les agences peuvent lister, modifier et supprimer des propriétés directement depuis la plateforme. De plus, ce projet est conçu pour être relié à une application mobile, qui n'est pas encore développée, pour offrir une plus grande accessibilité aux utilisateurs. Pour le développement de ce projet, j'ai utilisé Angular pour le front-end, en tirant parti de ses fonctionnalités puissantes pour créer une interface utilisateur interactive et réactive. Bootstrap a été utilisé pour le design et la mise en page, ce qui a permis d'obtenir un site web esthétiquement agréable et adaptatif. Pour le back-end, j'ai utilisé Node.js avec Express, ce qui a permis de créer un serveur web efficace et performant. Enfin, MongoDB a été utilisé pour la base de données, permettant de stocker et de récupérer des informations sur les propriétés. Ce projet m'a permis d'acquérir une expérience pratique dans le développement d'applications web full-stack et m'a donné une compréhension approfondie de la façon dont les différentes technologies peuvent être utilisées ensemble pour créer une solution fonctionnelle et conviviale.

Portfolio

Mon projet Portfolio le site sur lequel vous êtes actuellement. Il a été entièrement réalisé en HTML, CSS et JavaScript, trois technologies fondamentales du développement web. Le HTML a été utilisé pour structurer le contenu de la page, tandis que le CSS a permis de styliser et de mettre en page le site. J'ai utilisé JavaScript pour ajouter de l'interactivité à la page, comme des animations et des effets de transition. Ce projet m'a permis de mettre en pratique mes compétences en HTML, CSS et JavaScript, et de créer un site web responsive et interactif. Il m'a également donné l'occasion de montrer ma capacité à créer un design attrayant et une expérience utilisateur agréable. Je suis fier de ce projet car il représente mon parcours en tant que développeur web et met en valeur les compétences que j'ai acquises.
